# LavenderDragonDesign's PDF Generator

This repo contains everything you need to run the app locally and deploy it (e.g., GitHub + Netlify).

## Run Locally

**Prerequisites:** Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Build

`npm run build`
